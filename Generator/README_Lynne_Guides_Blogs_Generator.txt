Lynne Guides and Blogs Generator

Open Generator/Lynne_Guides_Blogs_Generator.html in a browser.

Use it to create a simple guide or blog page for The Listening Space Counselling.

How to use it:
1. Choose Guide or Blog.
2. Add the title, file name, short introduction, card summary and main page text.
3. Click Generate.
4. Click Download page and save the file in the folder shown on screen.
5. Copy the card snippet into guides.html or blog.html.

Notes:
- If the browser blocks the copy button, the generated text remains visible in the output box so it can be selected and copied manually.
- Guide pages should be saved inside the guides folder.
- Blog pages should be saved inside the blog folder.
- The file name should use lower case words with hyphens.
- After adding a new page and card, upload both changed files to GitHub.
